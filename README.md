# README Builder

> Create professional README files in minutes. Simple form-based interface for developers who want great documentation
> without the hassle.

![Platform](https://img.shields.io/badge/Platform-Windows-blue)
![KMP](https://img.shields.io/badge/Kotlin-Multiplatform-purple)

## ‚ú® Features

- **Smart Templates** - Pre-built templates for different project types (mobile apps, web apps, CLI tools, multiplatform
  projects, and more)
- **Dynamic Forms** - Form fields automatically adjust based on your project type
- **Real-time Preview** - See your README as you build it
- **Zero Configuration** - Install and start creating immediately
- **Export to Markdown** - One-click export to standard `.md` format

## üöÄ Getting Started

### Installation

1. Download the latest release for your operating system from the [Releases](../../releases) page
2. Run the installer
3. Launch README Builder
4. Start creating professional documentation!

### Usage

1. **Select your project type** from the dropdown menu
2. **Fill in the form** with your project details
3. **Preview in real-time** as you type
4. **Export** your README.md when you're done

That's it! No configuration needed.
